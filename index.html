Simon Game

Press ENTER to start the game

1️⃣ Start the Game. Press any key on the keyboard to start the game.
2️⃣ Watch the Sequence One of the colored buttons will flash. This is the start of the sequence.
3️⃣ Repeat the Pattern Click the same button that flashed.
4️⃣ Next Level After you correctly repeat the pattern, the game will: move to the next level add one more button to the sequence.
5️⃣ Remember the Order Each level the sequence becomes longer, so you must remember all previous colors in the correct order.
